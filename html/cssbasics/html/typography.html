Learn Typography

Inherit sample

When we apply rem, it will be applied only for few elements not for all the elements. if we want to apply the same for all the elements we can use the "inherit" value for font-size property

Text-Decoration

It is used to provide styles to the text. example values for the properties underline, overline

Text-transform

It is used to transform a text into a different form like capitalize, uppercase etc

Text-align

Align the text right, left, justify

Text-indent

It will indent the text from the header

Line-height

It is used to override the default value provided by browser, difference between two lines

Letter-spacing

It is used to provide space between characters

Font-family

It is defined to set the values for font. mostly we will be specifying multiple values as if one value is not present another values will be considered for the font. We need to define the default font
