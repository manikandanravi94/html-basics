Flex-box

It is used to display the elements in whatever the mode we want to show for example we can show the vertical elements in the horizontal mode. horizontal bar in the vertical mode. Below are the frequent properties used specific to flex-box

display: flex

align-items : center,flex-start, flex-end

justify-content : flex-start, flex-end,center,spacke-between, space around

flex-direction : row,column,row-reverse,column-reverse

flex-wrap : wrap, no-wrap, wrap-reverse

flex-grow, flex-shrink : value can be given in the form of (x)times

flex : (flex-grow) (flex-shrink) flex-basis Ex: 1 1 40px

flex-flow : (flex direction) (flex wrap)
