Css units and size

colors :

To set a colour of a text we can use the "color" property

To set the backgroud-color for the page we can use the "backgroud-color" property

We can use the coolors.io website to see the contrast check and other items

Different types to set color values are hex value,rgb,rgba,hs;,hue,hsa etc

Fonts :

we can use the font-size to set the value of the font.. by default "font-size:16px"

We should not change the font-size at root elements as it would affect the browser settings to change the font-size as per user wish

We can use "font"(shorthand property which lets you set multiple fonts at a same time) property to set the font

width:40ch; will help us to restrict the element values to be 40 chars.. exceeded chars will be moved to the next line

We can use relative properties like "em" refers parent element "rem" refers root element. values will get adjusted to the super element

We sometime required page height to be occupied fully. for that case we can use vh property to 100 instead of the height property we can use min-height as it vl enlarge the page as the content grows
